QUIC Deployment in Vietnam

Insights from our 2025 nationwide QUIC scanning study

Methodology Results Map Authors

Introduction

This project presents the first large-scale analysis of QUIC protocol deployment across Vietnam. Using tools like ZMap, ZDNS, and QScanner, we scanned over 16 million IPv4 addresses to assess QUIC adoption and visualize its distribution geographically.

Methodology

Our scanning framework integrates:

  • ZMap – for initial IP responsiveness over UDP:443
  • ZDNS – for resolving hostnames from IPs
  • QScanner – for full QUIC + TLS 1.3 handshakes
Each detected QUIC-enabled server was enriched with geolocation and AS data using IP2Location.

Results Overview

  • 8,337 QUIC-enabled IPs detected (≈0.05% of all scanned IPs in Vietnam)
  • Top ASes: VNPT, Viettel, FPT Telecom
  • QUIC version: 99.3% conform to IETF QUIC (RFC 9000)
  • Active migration: Supported by 69.4% of servers
  • TLS cipher suite: 80.5% use AES 128 GCM SHA256
